# とけてる値札メーカー

ブラウザで動く、ローカル完結の値札・ラベル作成ツールです。  
小さな値札を手書きする負担を減らして、**「印刷してすぐ使える」**形に寄せるために作りました。  
福祉施設・作業所・ハンドメイド販売・フリマ準備など、**細かい値札づくりが地味に大変な場面**を楽にすることを目指しています。

## できること

### 1. 値札モード
- 価格、通貨、税込 / +税 表示
- 商品名・補足の入力
- アイコン追加
- カラー変更
- テンプレート変更
- 背景パターン / 季節テーマ
- 用紙サイズ変更
- 値札サイズ変更
- PDF書き出し

### 2. A-ONEラベルモード
- A-ONEラベル用に並べて印刷
- 対応シート:
  - 31555（70面 / 20×20mm）
  - 73200（200面 / 20×8mm）
- 枠ガイド表示
- 位置微調整
- 位置合わせテスト印刷（枠のみ）

### 3. 管理番号モード
- 連番ラベルを自動生成
- プレフィックス設定（例: B-001）
- 開始番号変更
- 生成枚数変更
- 3桁 / 4桁ゼロ埋め
- A-ONEラベルにそのまま並べて印刷

## 特徴

- **インストール不要**  
  HTMLを開くだけで使えます。

- **ローカル完結**  
  入力した内容はブラウザのローカル保存を使っています。  
  サーバーやデータベースは使っていません。

- **スマホでも使いやすい**  
  拡大プレビューつきで、細かい値札も確認しやすくしています。

- **高画質PDF出力**  
  文字がにじみにくい高画質出力に対応しています。

- **現場向けの調整がしやすい**  
  「とりあえず印刷して試す」「少し位置をずらす」「貼るだけ運用に寄せる」など、実際の作業に合わせて調整しやすい構成です。

## こんな場面向け

- 作業所・福祉施設の販売準備
- ハンドメイド作品の値札づくり
- フリマ / イベント準備
- 小さな管理番号シール作成
- 手書き値札を減らしたい時

## 使い方

1. `index.html` をブラウザで開く
2. モードを選ぶ
   - 値札モード
   - A-ONEラベル
   - 管理番号
3. 内容を入力する
4. プレビューで確認する
5. PDF書き出しする
6. 100%（実寸）で印刷する

## 印刷のコツ

- A-ONEラベルは、最初に**テスト印刷（枠のみ）**で位置確認するのがおすすめです
- 印刷倍率は**100%（実寸）**にしてください
- ズレがある場合は、位置微調整で少しずつ合わせてください

## ファイル構成

- `index.html`  
  本体。HTML / CSS / JavaScript を1ファイルにまとめています。
